The 'Get Started' section outlines a 3-step user flow. Step 01 is Discover (browse agents), Step 02 is Build (install CLI), and Step 03 is Deploy (scale workflows). Each step includes a card with an icon, title, body copy, and a relevant code block or CTA. A secondary path for listing APIs is provided below the cards.
