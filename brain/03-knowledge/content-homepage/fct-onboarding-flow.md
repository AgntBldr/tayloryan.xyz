The 'Get Started' section on the Teneo homepage outlines a 3-step user flow under the headline 'Your first agent query in under 2 minutes.' Step 01 is Discover (browse 760+ live agents by capability), Step 02 is Build (install the CLI in one command), and Step 03 is Deploy (scale workflows or launch your own agent for USDC). Each step is a card with an icon, title, body copy, and a code block or CTA. A secondary 'List your API' path sits below the cards.
