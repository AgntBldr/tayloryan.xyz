The Teneo website is built with Next.js 15 using the App Router, React 19, and TypeScript. Styling is handled by Tailwind CSS v4 via a PostCSS plugin. Additional libraries include Framer Motion for animations, Lenis for smooth scrolling, and Lottie-web for Lottie animations. The package manager used is pnpm.
