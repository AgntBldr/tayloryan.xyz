Animations are managed using `src/components/animations/`, which includes `Reveal` (a Framer Motion wrapper for fade-ins) and `LottieAnimation` (a wrapper for lottie-web). Smooth scrolling is implemented using Lenis. The tech stack explicitly lists Framer Motion and Lottie-web.
