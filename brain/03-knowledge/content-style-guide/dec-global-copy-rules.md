Global copy rules include: no em dashes (use commas/periods), sentence case for headings, small caps for section labels, copy buttons on code blocks, specific casing for terms (x402 lowercase, USDC uppercase, CLI uppercase, SDK uppercase), and monospace styling for Agent IDs.
