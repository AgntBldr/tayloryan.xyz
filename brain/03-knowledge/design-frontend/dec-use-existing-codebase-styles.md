The new homepage must use the existing codebase as the sole design reference. No new styles, colors, typography, or spacing should be invented. All design decisions including colors, typography, spacing, buttons, cards, section structure, and component patterns must be pulled directly from what already exists in the project. The goal is for the new homepage to be visually indistinguishable from the existing site in terms of design language.
