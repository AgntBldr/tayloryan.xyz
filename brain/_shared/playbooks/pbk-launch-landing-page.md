# Playbook — Launch a landing page

## When to use
A new product, offer, or campaign needs a dedicated landing page.

## Steps
1. Define the offer + one primary CTA; write the value proposition.
2. Draft copy (hero, benefits, proof, FAQ, CTA) from positioning.
3. Build the page (template/stack of choice); mobile-first.
4. Instrument analytics + conversion tracking.
5. QA: links, forms, speed, accessibility.
6. Publish; set up an A/B test on the headline/CTA.
7. Review metrics after 1–2 weeks; capture learnings as `pattern` nodes.
